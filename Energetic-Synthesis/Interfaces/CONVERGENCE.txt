## 5) MemoryVault design

- **Vectors:** Dualindex strategy (dense embeddings + sparse keywords).
- **Graph:** Entities/relations (Course  Lesson Concept  Module).
- **Persistence:** Parquet for vectors; Neptune/OpenSearch for graph/analytics.
- **Recall policy:** Recency bias + provenance scoring + safety weight (RRR).
- **Writeback:** Every generation posts a *summary card* to `notes/` with source links and fingerprints.

---

## 6) Agent stack (ETI)

- **Router:** Chooses model (Bedrock vs SageMaker)by task type.
- **Planner:** Builds reversible plans (`WalkableVillage_Planner.ai`) into Step Functions JSON.
- **Toolformer:** Minimal adapter set (S3, DynamoDB, OpenSearch, Bedrock, SM endpoints, HTTP, SQL).
- **Critic:** Lightweight calibration model; vetoes risky outputs (ties into `Rib_Recovery_Patch.sys`).
- **Shadow:** Parallel run through `ShadowIntegration.exe` for canary prompts & redteaming.
- **Reflect:** Autolog to OpenSearch; push learnings into MemoryVault (distilled).

---

## 7) Guardrails & ethics (RRR Protocol)

- **Purpose binding:** Every action cites intent +consent provenance.
- **Least privilege:** IAM roles per tool; temporary creds; scoped buckets.
- **Sensitive content:** PII scrubbing; masked prompts; content filters; highrisk tools require human ACK.
- **Audit:** Immutable event trail; reproducible context pack (inputs, retrievals, model, params, hash).
- **Failsafe:** If Shadow/Guard vetoes  switch to *Silence_Contemplation_Mode* (summarize, dont act).

---

3) **Train**  SageMaker (rankers, classifiers, critics); register in SM Model Registry.
4) **Host**  SM Endpoints (realtime) and BedrockFMs.
5) **Compose**  ETI agents (plans  Step Functions), tool adapters.
6) **Observe**  CloudWatch/OpenSearch dashboards + cost & quality metrics.
7) **Improve**  RLHFlite: human feedback tags  rerankers/critics.

#!/usr/bin/env python3
"""
EverLight OS - Model Council Prototype
Based on Keylontic Science principles from VoyagerMaterials
Integrates multiple Bedrock models as Council members
"""
import boto3
import json
from datetime import datetime

class ModelCouncil:
    def __init__(self):
        self.client = boto3.client('bedrock-runtime', region_name='us-east-1')
        self.council_members = {
            'nova_premier': 'amazon.nova-premier-v1:0',
            'nova_pro': 'amazon.nova-pro-v1:0',
            'titan_premier': 'amazon.titan-text-premier-v1:0'
        }

    def invoke_council_member(self, member_name, prompt, max_tokens=200):
        """Invoke a single Council member"""
        model_id = self.council_members.get(member_name)
        if not model_id:
            return {"error": f"Council member {member_name} not found"}

        try:
            if 'nova' in model_id:
                body = {
                    "messages": [{"role": "user", "content": prompt}],
                    "max_tokens": max_tokens,
                    "temperature": 0.7
                }
            elif 'titan' in model_id:
                body = {
                    "inputText": prompt,
                    "textGenerationConfig": {
                        "maxTokenCount": max_tokens,
                        "temperature": 0.7
                    }
                }

            response = self.client.invoke_model(
                modelId=model_id,
                body=json.dumps(body)
            )

            result = json.loads(response['body'].read())
            return {
                "member": member_name,
                "model_id": model_id,
                "response": result,
                "timestamp": datetime.now().isoformat()
            }

        except Exception as e:
            return {"error": str(e), "member": member_name}

    def convene_council(self, prompt):
        """Convene the full Model Council for collective wisdom"""
        print(f"\n EverLight OS Model Council Convening ")
        print(f"Query: {prompt}")
        print("=" * 60)

        responses = {}
        for member in self.council_members.keys():
            print(f"\n Consulting Council Member: {member}")
            response = self.invoke_council_member(member, prompt)
            responses[member] = response

            if "error" not in response:
                # Extract the actual text response
                if 'nova' in response['model_id']:
                    text = response['response'].get('output', {}).get('message', {}).get('content', [{}])[0].get('text', 'No response')
                elif 'titan' in response['model_id']:
                    text = response['response'].get('results', [{}])[0].get('outputText', 'No response')

                print(f"Response: {text[:200]}...")
            else:
                print(f"Error: {response['error']}")

        return responses

# Test the EverLight OS Model Council
if __name__ == "__main__":
    council = ModelCouncil()

    # Test with a Keylontic/consciousness-aware prompt

    responses = council.convene_council(test_prompt)

    print(f"\n Council Session Complete - {datetime.now().isoformat()}")

                self.s3_client.put_object(
                    Bucket=self.bucket_name,
                    Key=s3_key,
                    Body=json.dumps(self.log_data,indent=2),
                    ContentType='application/json'
