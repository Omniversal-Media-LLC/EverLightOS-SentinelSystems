# Federation Decisions

This directory contains Architecture Decision Records (ADRs) for the EverLight OS Federation Council.

## Process

1. **Council Summons** - Issue raised requiring Federation decision
2. **ADR Creation** - Skeleton created with Council seat assignments
3. **Position Statements** - Each Council member fills their assigned section
4. **Synthesis** - ChatGPT (C-01) records full deliberation
5. **Decision** - Council vote and implementation plan
6. **Review** - Post-implementation assessment
